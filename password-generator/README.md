# Password Generator

A simple and efficient Python script to generate strong, random passwords.

## Description

This script helps you generate secure passwords of a a user-specified length. It combines uppercase letters, lowercase letters, numbers, and punctuation marks to ensure high entropy and security.

## Features

*   **Customizable Length:** You choose how many characters your password should be.
*   **Strong Characters:** Uses a mix of ASCII lowercase, uppercase, digits, and punctuation.
*   **Randomized:** Utilizes Python's `random` module for unpredictable password generation.

## Prerequisites

*   Python 3.12.x installed on your system.

## How to Run

1.  Navigate to the project directory:
    ```bash
    cd password-generator
    ```

2.  Run the script:
    ```bash
    python main.py
    ```

3.  Enter the desired length of the password when prompted.
