# Python Mini Projects

A collection of beginner-friendly **Python mini projects** built to strengthen core programming concepts such as conditionals, loops, functions, randomization, and user input handling.

This repository is ideal for:
- Beginners learning Python
- Practicing logic-building
- Building a strong GitHub portfolio

---

##  Projects Included

###  Password Generator
Generates strong, random passwords using letters, numbers, and special characters.

**Concepts Used:**
- `random` module
- String manipulation
- User input
- Loops and conditionals

 Folder: `password-generator`

---

### Perfect Guess Game
A number-guessing game where the user tries to guess a randomly generated number in the minimum number of attempts.

**Concepts Used:**
- While loops
- Conditionals
- Random number generation
- Basic game logic

 Folder: `perfect-guess-game`

---

### ✊✋✌️ Rock Paper Scissors
A classic Rock–Paper–Scissors game played between the user and the computer.

**Concepts Used:**
- Conditional logic
- Random choice
- User input validation
- Game outcome logic

 Folder: `rock-paper-scissor`

---

##  Technologies Used
- **Python 3**
- Standard Python libraries only (no external dependencies)
